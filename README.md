# LightC
 ## A sample oop&gc language.
 ## Support:
 * class definition
 * type coercion
 * garbage collection
 * type cast and polymorphism(not finish yet)
 * circular reference detection(not finish yet)
 ## Should not support:
 * class inherit
 ## Platform
  Currently aarch64&linux only, could easily extend to other platforms.
